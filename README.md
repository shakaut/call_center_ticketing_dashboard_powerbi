# Call Center Ticketing Management Dashboard (Power BI)

## Project Overview

This project is based on a **call center ticketing management portal dataset**.
The goal of this dashboard is to **analyze call volume, issue reasons, department performance, and executive productivity** using Power BI.

The dashboard helps management understand:

* How many calls/tickets are received
* What issues customers face most
* Which departments and projects receive the most queries
* How executives perform over time

---

## Dataset Description

The dataset contains call/ticket records with the following fields:

* Issue ID
* Category (issue type)
* Department
* Project
* Issuer name
* Service executive name
* Created date
* Status (Resolved)

The data was cleaned and transformed using **Power Query** before visualization.

---

## Key KPIs

* Total calls / tickets
* Calls by issue category
* Calls by project
* Calls by department
* Status-wise calls (Resolved)
* Executive-wise total calls handled

---

## Dashboard Features & Visuals

* Total calls overview
* Reason for calls (issue category analysis)
* Calls answered by service executives
* Project-wise enquiry analysis
* Department-wise call distribution
* Status-wise call tracking
* Executive-wise animated chart showing total calls handled
* Daily, monthly, and lifetime call trend charts

---

## Filters & Interactivity

The dashboard includes the following slicers:

* Year
* Month
* Executive name
* Category

These filters allow users to interactively explore the data and drill down into specific time periods or executives.
